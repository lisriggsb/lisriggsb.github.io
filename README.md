# Ellie Riggsbee — personal website

This is a small static personal website built with plain HTML, CSS, and
JavaScript. There is no build step.

## Main Files

- `index.html` — home page
- `about.html` — about page
- `research.html` — research page
- `teaching.html` — teaching page
- `fun.html` — fun/projects page
- `music.html` — music page
- `newcomb.html` — interactive Newcomb problem simulator
- `contact.html` — contact page
- `styles.css` — shared site styling
- `cursor.js` — optional star cursor effect
- `brainvat.js` — brain-in-a-vat page buddy
- `cv.pdf` — public CV

## Local Preview

From this folder, run:

```sh
python3 -m http.server 8766
```

Then open:

```txt
http://localhost:8766/
```

## Publishing

This site is intended to publish with GitHub Pages from the `main` branch, using
the repository root as the publishing source.
